# SieteyMedia
Juego 7 y media en php

## Requisitos basicos programas requeridos

- php
- git

## Descargar el programa

### Con git
con git nos situaremos en el escritorio

cd <ruta>/desktop

git clone https://github.com/DavidChaconRamirez/SieteyMedia

(sino tenemos el archivo u ejercicio)

## Acceder al directorio donde tenemos el proyecto desde el terminal:

cd <ruta>

## ejecutar el programa

php ejercicio.php

## Ejecución del Programa

El programa se ejecutara de forma automatica, 
cuando se le haya dado la carta al usuario se le preguntara si este quiere continuar o pasar.

Una vez terminada la partida el programa nos preguntara si queremos jugar de nuevo.

Al final nos mostrara la baraja que hemos usado poniendo.

x: cartas usadas
M: cartas no usadas
